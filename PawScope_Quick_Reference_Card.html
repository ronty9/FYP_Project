📋 Application Overview

A comprehensive pet management & health monitoring app built with Flutter + Firebase. Features AI-powered breed identification and skin disease detection, schedule management with local notifications, and a full admin dashboard.

Flutter / Dart FastAPI / Python Firebase Docker / Cloud Run Node.js Cloud Functions
Flutter AppDart • MVVM
↕
FastAPIAI Backend
FirebaseBaaS
👤 User Features
  • Pet Profiles — Create & manage profiles, photo galleries, breed info
  • AI Breed Scan — Identify breed from photo (top-5 predictions)
  • AI Disease Scan — Detect skin diseases (top-3 predictions)
  • Scan History — Past results with confidence scores
  • Schedules — Create/edit care reminders with calendar view
  • Notifications — Local push reminders, configurable sound & vibration
  • Feedback — Submit feedback; admins review & reply
  • Community Tips — Curated pet care knowledge base
  • Auth — Email/password + Google Sign-In + OTP verification
🛡️ Admin Features
  • Dashboard — Centralised usage & activity overview
  • User Management — View, suspend, modify user accounts
  • Feedback Mgmt — Read, reply, manage submissions
  • FAQ / Tips Mgmt — CRUD for FAQs & community tips
  • Analysis Records — Review all AI scan records
  • Activity Logs — Immutable audit trail (logins, suspensions, critical actions)
⚠ Note: Activity logs are immutable — they cannot be edited or deleted once recorded.

PawScope Setup & Commands

Quick Reference
✅ Prerequisites
  • Flutter SDK — ^3.9.2
  • Dart SDK — bundled with Flutter
  • Python — 3.11+ (AI backend)
  • Node.js — v22 (Cloud Functions)
  • Firebase CLI — npm i -g firebase-tools
  • IDE — VS Code / Android Studio / IntelliJ
📱 1. Flutter App Setup
# Clone & install
$ git clone <repo-url>
$ cd FYP_Project
$ flutter pub get

# Ensure Firebase config files are in place:
# Android → android/app/google-services.json
# iOS → ios/Runner/GoogleService-Info.plist
# API Keys (git-ignored)
$ cp lib/config/api_keys.example.dart \
     lib/config/api_keys.dart
# Then update with actual SendGrid API key
🤖 2. AI Backend Setup
# Create virtual environment
$ cd backend
$ python3 -m venv .venv
$ source .venv/bin/activate

# Install dependencies
$ pip install -r requirements.txt

# Start dev server
$ uvicorn main:app \
    --host 0.0.0.0 --port 8000 --reload
# Or use: ./start.sh
⚡ 3. Cloud Functions
$ cd functions
$ npm install

# Deploy to Firebase
$ firebase deploy --only functions

# Or run locally
$ firebase emulators:start --only functions
Function Purpose
resetPassword Server-side password reset
sendOtp OTP email delivery via SendGrid
🚀 4. Firebase Rules & Deploy
# Deploy Firestore rules
$ firebase deploy --only firestore:rules

# Deploy Firestore indexes
$ firebase deploy --only firestore:indexes

# Deploy Storage rules
$ firebase deploy --only storage

# Run the app
$ flutter run
🐳 AI Backend — Cloud Run Deployment
Option A: Google Cloud Run
$ cd backend
$ gcloud run deploy petscan-ai \
    --source . \
    --region asia-southeast1 \
    --allow-unauthenticated
Option B: Docker (Local)
$ docker build -t petscan-ai ./backend
$ docker run -p 8080:8080 petscan-ai
